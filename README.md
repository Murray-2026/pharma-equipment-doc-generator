# 🏭 制药设备售前文档生成器

一款基于Streamlit的制药设备售前文档自动生成工具，帮助销售人员快速生成专业的技术文档、产品介绍和招标参数。

## ✨ 功能特点

- **多设备类型支持**: 支持粉碎机、混合机、制粒机、压片机、包装机等多种制药设备
- **多种文档类型**: 可生成技术方案书、产品介绍、招标参数、报价单等文档
- **自动参数填充**: 内置制药设备专业参数模板，快速填充技术规格
- **认证合规**: 支持GMP、CE、FDA、ISO9001等认证要求的文档生成
- **一键下载**: 生成专业的Word文档，方便编辑和分享

## 🚀 快速开始

### 本地运行

1. 克隆此仓库：
```bash
git clone https://github.com/YOUR_USERNAME/pharma-equipment-doc-generator.git
cd pharma-equipment-doc-generator
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行应用：
```bash
streamlit run app.py
```

4. 在浏览器中打开 http://localhost:8501

### 云端部署

一键部署到Streamlit Cloud：

[![Deploy to Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/deploy)

## 📋 使用流程

1. **选择设备类型**: 从下拉菜单中选择要生成文档的设备类型
2. **选择文档类型**: 选择需要生成的文档格式
3. **填写项目信息**: 输入项目名称和客户名称
4. **输入设备参数**: 填写产能、功率、电压、尺寸等技术参数
5. **选择认证要求**: 选择需要的认证类型
6. **生成文档**: 点击"生成文档"按钮，系统将自动生成专业文档
7. **下载使用**: 点击"下载文档"按钮获取Word文档
